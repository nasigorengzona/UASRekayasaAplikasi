# UASRekayasaAplikasi

## Overview 
This Web API gives list of makeup products information using : http://makeup-api.herokuapp.com

## Requirement 
- NodeJs 

## Avaible API
- GET / brand
- GET / tags
- GET / proudctTypes

## Dependencies
- Express 
- Axios

## Run Project :
Using NPM 
- Clone this repo
- Install Dependencies 
- Using console start index.js
